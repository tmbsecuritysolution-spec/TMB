# TMB Security Solutions — Virtual Business Card

A premium, mobile-first virtual business card for **TMB Security Solutions**. Built with modern HTML5, CSS3 (Glassmorphism), and Vanilla JavaScript.

## 🔗 Live Demo
[View the Live Card](https://yassine6815.github.io/virtual-card-deep-tech/)

## ✨ Features
- **Dynamic Status**: Automatically shows "Open" or "Closed" based on business hours.
- **Multilingual**: Supports English and Arabic with proper RTL support.
- **Branch Support**: dedicated sections for different locations (Laghouat, Aflou).
- **One-Tap Actions**: WhatsApp, Call, and Email integration.
- **SEO Optimized**: Standard meta tags and Open Graph support for social sharing.
- **PWA Ready**: Icons and manifest for "Add to Home Screen" support.

## 🚀 Deployment to GitHub Pages
1. Push this repository to GitHub.
2. Go to **Settings** > **Pages**.
3. Under **Build and deployment**, set Source to **Deploy from a branch**.
4. Select the `main` branch and `/ (root)` folder.
5. Click **Save**.
